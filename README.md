## 🛒 Sistema de Comparação de Preços

Aplicação web para monitorar e comparar preços de produtos em diferentes marketplaces como Amazon, Mercado Livre, Casas Bahia e Magalu.

O sistema permite cadastrar produtos por meio de links, acompanhar variações de preço ao longo do tempo e receber notificações quando os valores atingirem a faixa desejada.
